# Telegram Bot для каталога банных чанов

Бот для управления каталогом банных чанов с возможностью добавления товаров.

## 🚀 Возможности

- **Создание категорий (чанов)** - добавление различных видов банных чанов
- **Добавление товаров** - товары с названием, описанием, ценой и фото
- **Просмотр каталога** - навигация по категориям и товарам
- **Удаление** - удаление категорий и товаров
- **SQLite база данных** - хранение всех данных

## 📋 Требования

- Python 3.8+
- aiogram 3.x
- aiosqlite

## 🔧 Установка

1. Установите зависимости:
```bash
pip install aiogram aiosqlite
```

2. Получите токен бота:
   - Откройте @BotFather в Telegram
   - Создайте нового бота командой `/newbot`
   - Скопируйте полученный токен

3. Вставьте токен в файл `banya_bot.py`:
```python
BOT_TOKEN = "ваш_токен_здесь"
```

## 🎯 Запуск

```bash
python banya_bot.py
```

## 📖 Использование

### Команды:
- `/start` - Главное меню
- `/help` - Справка

### Функции через меню:

1. **📋 Каталог чанов** - просмотр всех категорий
2. **➕ Добавить чан** - создание новой категории
3. **➕ Добавить товар** - добавление товара в категорию
4. **🗑 Удалить чан** - удаление категории со всеми товарами

### Добавление чана (категории):
1. Нажмите "➕ Добавить чан"
2. Введите название чана
3. Введите описание (или `-` чтобы пропустить)

### Добавление товара:
1. Нажмите "➕ Добавить товар"
2. Выберите категорию
3. Введите название товара
4. Введите описание (или `-` чтобы пропустить)
5. Введите цену (или `0` чтобы не указывать)
6. Отправьте фото (или `-` чтобы пропустить)

## 🗄 Структура базы данных

### Таблица `categories`:
- `id` - уникальный идентификатор
- `name` - название чана
- `description` - описание
- `created_at` - дата создания

### Таблица `products`:
- `id` - уникальный идентификатор
- `category_id` - ссылка на категорию
- `name` - название товара
- `description` - описание
- `price` - цена
- `photo_file_id` - ID фото в Telegram
- `created_at` - дата создания

## 📝 Примеры

### Создание чана "Классический чан":
```
Название: Классический чан
Описание: Традиционный банный чан из дуба

Категория: Классический чан
Название: Веник дубовый
Описание: Качественный дубовый веник
Цена: 500
Фото: [отправить фото]
```

## ⚙️ Настройка

```python

```

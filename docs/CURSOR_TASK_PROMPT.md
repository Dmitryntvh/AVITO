# Ready-to-use prompt for Cursor/Codex/Replit

Нужно создать MVP внутреннего сервиса генерации карточек товара для банных чанов.

Стек:
- Python 3.11+
- FastAPI
- Pillow
- SQLite
- React + Vite

Функции MVP:
1. CRUD товаров
2. Загрузка фото товара
3. Хранение параметров товара
4. JSON-шаблоны слайдов
5. Генерация 8 PNG карточек из шаблонов
6. Скачивание ZIP
7. Превью результатов в интерфейсе

Ключевое:
- фото товара вставлять без искажения пропорций
- текст рендерить шрифтами, а не картинкой
- все координаты и стили шаблонов хранить в JSON
- обеспечить единый стиль карточек 1920x1440 (4:3)
- safe zone 200 px

Ожидаемый результат:
- структура проекта,
- API,
- frontend,
- локально запускаемый рабочий MVP.
